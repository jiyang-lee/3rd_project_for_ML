# HeatGridAgent 문서 정리본

`C:\3rd_Project\HeatGridAgent`는 `C:\3rd_Project\MyPro\HetGridCopilot`와 `C:\3rd_Project\Third_Idea`에 흩어져 있던 HeatGrid 관련 문서를 한곳에 모은 정리 폴더다.

- `03_전략_기획`: 디벨롭 방향, 통합 포지셔닝, 실행 전략
- `04_리서치`: 데이터셋 요약, 출시 타당성 검토
- `05_데이터셋`: PreDist 원본 zip과 메타데이터
- `99_참고자료`: 공통 가이드와 이전 참고 문서
