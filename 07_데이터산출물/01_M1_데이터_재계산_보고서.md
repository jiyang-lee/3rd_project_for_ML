# M1 데이터 재계산 보고서

## 기준

- 대상: `manufacturer_1`만 사용
- 원본 운영 데이터: ZIP 내부 `manufacturer 1/operational_data/substation_*.csv`
- 메타데이터: `metadata/manufacturer_1/normal_events.csv`, `metadata/manufacturer_1/faults.csv`
- 제외: 다른 제조사 그룹, `disturbances.csv`, strict 조건 미달 fault

## 라벨 재계산

| label | 건수 | window 기준 |
|---|---:|---|
| `normal` | 35 | `Event start`부터 `Event end`까지 |
| `efd_possible` | 15 | `Report date` 직전 7일 |

- normal window 길이: [7.0]
- positive window 길이: [7.0]
- strict positive Event ID: [1, 3, 7, 10, 15, 20, 40, 44, 47, 49, 52, 53, 57, 64, 67]

## 7일 window 커버리지

| label | coverage min | coverage median | coverage max |
|---|---:|---:|---:|
| `normal` | 1.0000 | 1.0000 | 1.0000 |
| `efd_possible` | 0.7242 | 1.0000 | 1.0000 |

- `efd_possible` Event ID 20은 7일 window 안의 실제 센서 row가 730개라 커버리지가 0.7242다.
- 해당 row는 제외하지 않고 `sample_count`, `coverage_rate`로 품질 정보를 남긴다.

## M1 공통 센서 컬럼

M1의 35개 운영 CSV에 모두 존재하는 컬럼만 사용했다. 일부 기계실에만 있는 급탕/세부 회로 컬럼은 이번 기준선에서 제외했다.

| 컬럼 | 설명 |
|---|---|
| `outdoor_temperature` | Outdoor temperature |
| `s_hc1_supply_temperature` | Heat circuit 1 flow temperature (secondary) |
| `s_hc1_supply_temperature_setpoint` | Heat circuit 1 reference flow temperature (secondary) |
| `p_hc1_return_temperature` | Heat circuit 1 return temperature (primary side) |
| `p_net_meter_energy` | Energy |
| `p_net_meter_volume` | Volume |
| `p_net_meter_heat_power` | Power |
| `p_net_meter_flow` | Flow |
| `p_net_supply_temperature` | Primary flow temperature |
| `p_net_return_temperature` | Primary return temperature |
