아키텍처 흐름

Agent 그래프

ops_graph는 스케줄러 틱에서 LLM을 호출하지 않는다. card_graph는 카드 버튼 클릭 시 캐시 확인, 근거 수집, 템플릿 또는 gpt-4o-mini 응답 생성, 티켓 기록을 수행한다.

실행과 검증

uv run python ../setup/bootstrap.py로 handoff와 replay 데이터를 준비하고, uv run pytest ../tests -q로 계약 테스트를 실행한다. 프론트는 npm run build와 npm audit --audit-level=moderate로 확인한다.

  • 백엔드 테스트 21개 통과, FastAPI TestClient의 httpx2 전환 경고 1건
  • RAG dry-run chunk 266개 확인
  • Docker DB public table 15개와 vector extension 확인
  • 데스크톱/모바일 viewport에서 주요 텍스트 겹침과 수평 overflow 없음

일부 원본 PDF는 텍스트 레이어 인코딩 때문에 RAG 추출문이 깨질 수 있어, OCR 또는 원본 PDF 보정 전까지 HTML 문서와 정상 추출 PDF에 더 의존한다.
