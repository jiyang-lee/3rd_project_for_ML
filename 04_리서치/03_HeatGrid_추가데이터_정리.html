HeatGrid 추가 데이터 후보 정리

HeatGrid 단독 관점에서 PreDist를 중심 학습셋으로 유지하되, 접근 제한이 있거나 실사용 시 API 요청이 필요한 데이터는 제외하고, 브라우저에서 바로 접근 가능한 공개 데이터만 추렸다. 목적은 학습셋, 수요/영향도, 공급망/인프라, 운영 리스크 보강 데이터를 한눈에 검토하는 것이다.

작성 기준일: 2026-06-20 산출 형식: 정적 HTML 포함 기준: 공개 접근 가능 + 직접 링크 가능 제외 기준: 접근 제한 + API 키/요청 필요

정리 기준

  • 포함: Zenodo, Mendeley Data, 일반 공개 웹페이지, 공공데이터포털의 직접 다운로드형 파일데이터
  • 제외: 연구자 문의가 필요한 제한 데이터, 활용신청·인증키가 필요한 OpenAPI, 런타임 호출 전제 데이터
  • 분류 규칙: HeatGrid 설명과 운영 판단 정확도에 직접 기여하면 필수 요소, 일반화 검증·배경 보강 성격이면 보강 요소
